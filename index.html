Adriana's Notes THE BASICS OF THE WEB AND HTML

1. Introduction to the Web

World Wide Web is a collection of HTML documents.

HTML stands for HyperText Markup Language, this is the basis for the majority of web pages.
HTML is what connects most types of documents together through hyperlinks.
HTML can be made up of text content (what you see), markup (arrangement of content), references to other documents, and links to other pages.

HTML markup is made out of tags. A tag is formed by a less than symbol, the tag name, a greater than symbol, the tag contents, followed by a less than symbol, a slash, the tag name, and a greater than symbol.
The tag before the content is an opening tag and the one after is a closing tag. The whole structure is called an element.

2. Computers are Stupid

Programmers need to write in a language a computer understands; this way a computer can follow the instructions the programmer wrote.
Therefore, in order to be precised, it is imperative to use the right syntax.

3. HTML Attributes

Anchor tags: adds a link to a text or word. When clicked, it directs you to another page.
Image tags: adds a link to an image. Another page is opened when the image is linked.
Whitespace: text on multiple lines using the break tag (void tag) or the paragraph tag (not a void tag).
Inline/Block: an inline tag tells when a line ends, while a block tag creates an invisible box.
Examples of inline tags include the anchor, span, break, image, and strong tags. While the paragraph and div tags are types of block tags.

4. Document Structure

The basic structure of an HTML document includes:
a DOCTYPE in the first line,
an opening HTML tag on the second line,
an opening and closing head tags, which will include title tags,
an opening an closing body tags, which will include the page's content,
and a closing html tag.
